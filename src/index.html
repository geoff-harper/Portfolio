YellowPages Site Viewer

This site previewer was something I developed in my spare time at YellowPages. Its purpose is to compile screenshots of some sites we created. This was done with vanilla CSS and Javascript so no particularly fancy tech was used. I wanted the aesthetic to simulate flipping through a box of vinyls which I tried to achieve with 3D transforms and box shadows. All of the previews themselves were designed and built by myself on our proprietary WYSIWYG platform. It should be noted that all the previews are drafts and belong to YellowPages™.
